### HDFS 基本概念
`HDFS`（Hadoop Distributed File System ，Hadoop分布式文件系统）是 Hadoop 的分布式文件系统，主要用来解决海量数据的存储问题，具有高容错、高吞吐量等特性，可以部署在低成本的硬件上。
